QUICK START:

  1. Create USB installer:
     sudo dd if=touchos-installer.iso of=/dev/sdX bs=4M status=progress

  2. Boot from USB:
     Insert USB → Restart → Press F12 → Select USB drive

  3. Install:
     Select "Install TouchOS" from boot menu
     Touch installer will launch with hardware detection

  4. First boot:
     Remove USB → Reboot → TouchOS loads from hard drive
